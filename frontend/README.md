# E-commerce Product Management System UI

React admin dashboard for managing products, categories, and inventory. The UI provides CRUD forms, table views, and local product image previews (preview only, not persisted).

## Features

- Product, category, and inventory CRUD screens
- Table-based admin views
- Local product image preview
- Error and loading states

## Tech Stack

- React
- Vite
- Plain CSS

## Getting Started

1. Install dependencies:
	- npm install
2. (Optional) Set API URL in frontend/.env:
	- VITE_API_URL=http://localhost:8000/api/v1
3. Start the dev server:
	- npm run dev

## Notes

- Image uploads are preview-only in the UI; they are not sent to the backend.
